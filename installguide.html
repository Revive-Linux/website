How to install ReviveLinux!

Step 1, Download the Arch Linux 32 ISO from This Link

Step 2. After the iso has downloaded, use your favorite flashing software (Rufus, Ventoy, balenaEther, dd) to flash the installer to a USB stick.

Step 3. After flashing the ISO to either a CD or USB, plug it into the olc computer and boot off of the CD/USB.

Step 4. After booting, run lsblk to list the hard drive. (it should be sda or sdb)

Step 5. Use cfdisk to delete any existing partitions off the hard drive. After deleting all of them, make a new one. After making the new one, write the changes and then exit cfdisk.

Step 6.After exiting cfdisk, run mkfs.ext4 /dev/sdx1 (replace x with your drive letter)

Step 7. Now mount the hard drive by running mount /dev/sdx1 /mnt (replace x with your hard drive letter)

Step 8. Now run wget http://revivelinux.nexalinux.xyz/install.sh To download the installer.

Step 9. Run chmod +x install.sh to make the installer executable.

Step 10. Run ./install.sh to begin the installation.

Step 11. After the install inished, run umount -a to unmount the hard drive.

Step 12. Run reboot to finish the installation.

If you have done everything right, you should see GRUB and the option to start Revive Linux.
